=== Features

==== Web Interface
* Windows 10 style: Modern blue with sad face emoji
* Windows 11 style: Refined design with subtle patterns
* Windows 7/8 style: Classic monospace with progress bar
* Windows XP style: Retro DOS font with CRT effects
* Responsive design for mobile devices
* Interactive progress bars that slowly increase
* Keyboard shortcuts for style switching
* Easter eggs (Konami code, style switchers)
* Customizable via URL parameters (style, code, message, percentage, QR)

==== REST API
* `GET /` - Display BSOD with customization options
* `GET /random` - Redirect to random BSOD style
* `GET /api/error` - Get random error data
* `GET /api/error/:code` - Get specific error by code
* `GET /api/codes` - List all available error codes
* `GET /api/styles` - List all available styles
* `GET /api/analytics` - Get usage statistics
* `POST /api/analytics/reset` - Reset analytics (dev only)
* `GET /api/health` - Health check endpoint
* `GET /api/metrics` - Prometheus metrics export

==== Error Messages
* Standard Windows errors (CRITICAL_PROCESS_DIED, etc.)
* Developer humor errors (COFFEE_NOT_FOUND, NPM_INSTALL_TIMEOUT, etc.)
* COFFEE_NOT_FOUND - Critical system resource missing
* PRODUCTION_DEPLOYMENT_ON_FRIDAY - The cardinal sin
* NPM_INSTALL_TIMEOUT - Heat death of universe expected
* STACKOVERFLOW_COPY_PASTE_ERROR - Malicious solutions detected
* MERGE_CONFLICT_ANXIETY - Unresolved conflicts
* UNDEFINED_IS_NOT_A_FUNCTION - Classic JavaScript error
* And 20+ more!

== Upgrade Guide

=== From 0.1.0 to 1.0.0

This is the first production release. If you had the initial skeleton:

1. Pull latest changes
2. Run `npm install` to install all dependencies
3. Copy `.env.example` to `.env` and configure
4. Run `npm start` to start the server
5. Visit `http://localhost:3000` to see the BSOD
